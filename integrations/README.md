# Integrations

This directory contains host-specific integrations that sit on top of the core `brain-tree-os` CLI.

- `openclaude/` contains the OpenClaude plugin and its documentation.
- `codex/` contains the Codex plugin source and registry scaffolding used for local installation.

These integrations are intentionally separate from `packages/cli/`, which contains the CLI itself and command-file variants for Claude-compatible hosts.
